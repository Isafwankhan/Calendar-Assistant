# Calendar-Assistant
Calendar Assistant is a streamlined, AI-enhanced productivity tool built to simplify how you interact with your digital life. Featuring a bold "Crimson" dark-mode aesthetic, it transforms the standard calendar experience into a high-level executive dashboard.

Calendar Assistant is a smart scheduler that syncs with Google Calendar and uses AI to summarize your day.

## Features
- **AI Insights:** Summarizes your busy schedule using Gemini 1.5 Flash.
- **Crimson UI:** Premium dark-mode interface.
- **All-Day Events:** Keep your tasks visible all day long.

## Installation
1. Clone the repo: `git clone https://github.com/yourname/Calendar-Assistant.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Add your `.env` file with `GEMINI_API_KEY`.
4. Place your `credentials.json` in the root folder.
5. Run the app: `streamlit run app.py`
